RPG experiment

My first real video game project, featuring robots and trees.

For this project, I wanted to try programming a simple RPG to learn about different mechanics in Unity. It was one of my first real game project. In retrospect, it was a projet way too large for a single developper, but I can definitely say I learned a lot making it, even though it is not finished. Here are some of the mechanics I implemented:

Character movement

The robot character I made for this game had no animations and only 2 poses; one with the arms raised to hold a weapon, one with arms down. When the player would press 'E', the character's pose would lerp between the two poses. The player movement was really simple; its position would be changed by the ASWD keys, and the character would lean towards the direction of motion. Also, its orientation would face where the mouse would point.

Inventory

I also implemented an inventory system where you could store weapons, armors and miscellaneous items. You could see the stats of all items, and you could put some items in a favorite spot (the bottom right circle) that you could access by pressing Control instead of opening the entire inventory.

Saving system

I implemented a saving system where the game would save the player's position, inventory and stats, and could load it on demand. It would start by storing all data inside a serializable Save class, and then save all information into a binary file. The binary file could be deserialized to load the game.
