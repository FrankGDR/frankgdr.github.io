Realistic realtime render with Unity

A tentative for realistic realtime rendering of a small studio apartment using Unity, including the modelling and texturing of all assets.

Interested by computer graphics as well as video game design, I wanted to try making a scene as realistic as I could. I also wanted to practice my 3D modelling skills using Blender, and try texturing with Substance Painter. I decided to model a small appartment with various furniture pieces and try to make it as realistic as possible using only assets I could create by myself. To achieve this, I had to read a lot of Unity documentation about lighting and watch a few tutorials about texturing in Substance Painter, which wasn't something I had done a lot.

For this scene, I used mixed global lighting (so a mix of baked and realtime lighting) as well as many reflective probes and a bunch of post processing effects (like bloom, a subtle vignette and ambiant occlusion). For now, I only modelled the living room and the kitchen. Altough I still have a lot to learn, I am quite pleased with the result, which yo ucan see below.
